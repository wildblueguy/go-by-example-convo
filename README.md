# go-by-example-convo
Series of Go language exercises (and answers) based on https://gobyexample.com, to serve as a tutorial or refresher. Exercises generated and proctored by ChatGPT.

- Exercise source: https://gobyexample.com
- Conversation dev environment: ChatGPT
- Code dev environment: https://go.dev/play or local

There are 25 total exercises, presented in 5 parts:

1. Basic Syntax
2. Data Structures & Functions
3. Pointers, Structs, Interfaces
4. Concurrency
5. Standard Library

Trivial exercises have been intentionally omitted. **To work all exercises yourself, go to https://chatgpt.com and provide the below prompt, additionally specifying that the exercises are to be administered one-by-one and that each answer should be the contents of a `.go` file, as in this repo.**

## Original prompt

> please generate an interactive graded coding exercise using **all** content from https://gobyexample.com as the source of correctness. present it to me as a neutral proctor that provides no assistance, but allow me to separately address an expert assistant on demand if i cannot answer on my own

(Follow-up prompts to improve format)
